# getYoutubeUrl ユーザーマニュアル（日本語）

YouTube を検索し、曲の再生・歌詞表示・MP3/MV ダウンロードを一つのウィンドウで行える GUI アプリです。  
YouTube API キーは不要です。

## インストールと起動

### macOS

```bash
cd getYoutubeUrl
./setup-mac.sh   # 初回のみ（Python, VLC, ffmpeg）
./run.sh
```

### Linux（Raspberry Pi など）

```bash
cd getYoutubeUrl
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
sudo apt install -y python3-tk vlc ffmpeg
./run.sh
```

### Windows

1. `setup-windows.bat` を実行（winget が無い場合は自動で手動インストールに切り替わります）
2. `run-windows.bat` で起動

> **必要なもの:** インターネット接続、VLC、MP3/MV 保存時は ffmpeg

| 領域 | 説明 |
|------|------|
| 上部 | 検索種類（曲 / MV）、検索語、件数、検索ボタン |
| 検索結果 | YouTube 検索結果の一覧 |

| 右パネル | 歌詞表示 |

## 言語の変更

1. 検索結果の下にある **「言語」** コンボボックスをクリック
2. **日本語 / 中文 / 한국어 / English** から選択

表示順: **日本語 → 中文 → 한국어 → English**

**既定言語は日本語** です。起動時から日本語 UI で表示されます。  
メニュー・ボタン・状態メッセージ・ダイアログがすべて選択した言語に切り替わります。

## 検索

### 検索種類

| モード | 説明 |
|--------|------|
| **🎵 曲検索** | 通常の楽曲を優先して検索（MV タイトルは後回し） |
| **🎬 ミュージックビデオ** | 「検索語 + official mv」で MV を優先検索 |

### 手順

1. **検索語** に曲名・アーティスト名を入力
2. **件数** で取得件数を指定（1〜200、既定 20）
3. **検索** をクリック、または `Enter` キー

### 検索結果の操作

| 操作 | 機能 |
|------|------|
| **追加 ↓** | 選択行を再生リストに追加（URL 重複はスキップ） |
| **🗑 削除** | 選択行を検索結果から削除 |
| **🎬 MV再生** | 選択行を MV ポップアップで再生 |
| **解像度** | MV 再生・保存時の最大解像度（HD / FHD / QHD / 2K / 4K） |
| **ブラウザで開く** | 既定ブラウザで YouTube を開く |
| **ダブルクリック** | 曲 → リスト追加 / MV → MV 再生 |

## 再生リストと再生

検索結果から **追加 ↓** で曲を溜め、順番に再生できます。  
リストは **曲数無制限** で、複数回の検索結果を混ぜられます。

| ボタン | 機能 |
|--------|------|
| **▶ 再生** | 選択曲を再生（🎵 は音声、🎬 は MV ポップアップ） |
| **⏸ 一時停止** | 再生 ↔ 一時停止 |
| **⏹ 停止** | 停止 |
| **⏭ 次へ** | 次の曲へ（ランダム ON 時は無作為） |
| **🔀 ランダム再生** | ランダム ON にして即ランダム再生 |
| **ランダム: オフ/オン** | 次曲・自動次曲のランダム切替 |
| **🗑 削除** | 選択曲をリストから削除 |
| **🗑 すべて削除** | リスト全消去 |
| **URLをすべてコピー** | リスト内 URL をクリップボードにコピー |

**▶** マーク付きの行が現在再生中の曲です。

## ミュージックビデオ（MV）

MV は **別ウィンドウ**（初期 800×600）で再生されます。

| 項目 | 内容 |
|------|------|
| 解像度 | 検索結果横の **解像度** で HD〜4K を選択（既定 FHD） |
| 全画面 | **F11** または映像ダブルクリック |
| 閉じる | **Esc**（全画面中は全画面解除） |
| 備考 | 高解像度は ffmpeg で映像+音声を結合して再生 |

MV 再生中はメインウィンドウの音声再生は自動停止します。

## MP3 / MV のダウンロード

| ボタン | 機能 |
|--------|------|
| **⬇ MP3ダウンロード (全体)** | 再生リストの全曲を MP3（192kbps）で保存 |
| **⬇ 選択曲保存** | 選択した 1 曲のみ MP3 保存 |
| **⬇ 選択MV保存** | 選択した MV を MP4 で保存（選択した解像度） |
| **⬇ MVダウンロード (全体)** | リスト内の全 MV を MP4 で一括保存 |

保存時にフォルダ選択ダイアログが開き、下部の状態行に進捗が表示されます。

---

## ローカル MP3

1. **📁 フォルダ選択** で MP3 フォルダを指定
2. 一覧から曲を選び **▶ MP3再生** またはダブルクリック
3. **すべて追加** で再生リストに一括追加

ローカル曲は **💾 ローカル** として表示されます。

---

## 歌詞パネル

右側の **歌詞** パネルに、再生中の曲の歌詞を表示します（`syncedlyrics` 使用）。

- 再生開始後に自動検索
- 見つからない場合は「歌詞が見つかりませんでした。」と表示
- `syncedlyrics` 未インストール時は歌詞機能のみ無効

## ショートカットキー

| キー | 動作 | 対象 |
|------|------|------|
| `Enter` | 検索実行 | メインウィンドウ |
| `F11` | 全画面切替 | MV ポップアップ |
| `Esc` | 全画面解除 / ポップアップ閉じる | MV ポップアップ |

## トラブルシューティング

| 症状 | 対処 |
|------|------|
| 検索できない | `.venv/bin/pip install -U yt-dlp` |
| 再生できない | VLC がインストールされているか確認（macOS: `~/Applications/VLC.app`） |
| MP3/MV 保存失敗 | ffmpeg をインストール |
| 歌詞が出ない | `pip install syncedlyrics` |
| 一部動画が失敗 | 地域制限・YouTube 側の制限の可能性 |
