# getYoutubeUrl 用户手册（中文）

一款基于 Python + tkinter + yt-dlp + libVLC 的 GUI 程序，可在同一窗口内完成 YouTube 搜索、播放、歌词显示以及 MP3/MV 下载。  
无需 YouTube API 密钥。

## 安装与运行

### macOS

```bash
cd getYoutubeUrl
./setup-mac.sh   # 首次运行（安装 Python、VLC、ffmpeg）
./run.sh
```

### Linux（Raspberry Pi 等）

```bash
cd getYoutubeUrl
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
sudo apt install -y python3-tk vlc ffmpeg
./run.sh
```

### Windows

1. 运行 `setup-windows.bat`（若无 winget 会自动切换到手动安装）
2. 运行 `run-windows.bat` 启动程序

> **依赖：** 网络连接、VLC；保存 MP3/MV 时需要 ffmpeg

## 切换语言

1. 点击搜索结果下方的 **「语言」** 下拉框
2. 选择 **日本語 / 中文 / 한국어 / English**

显示顺序：**日本語 → 中文 → 한국어 → English**

程序**默认语言为日语**。若需中文界面，请在语言框中选择 **中文**。  
所有菜单、按钮、状态提示和对话框都会立即切换。

## 搜索

### 搜索类型

| 模式 | 说明 |
|------|------|
| **🎵 歌曲搜索** | 优先搜索普通歌曲（MV 标题靠后） |
| **🎬 音乐视频** | 以「搜索词 + official mv」优先搜索 MV |

### 步骤

1. 在 **搜索词** 输入歌曲名或歌手名
2. 在 **数量** 设置结果数（1–200，默认 20）
3. 点击 **搜索** 或按 `Enter`

### 搜索结果操作

| 操作 | 功能 |
|------|------|
| **添加 ↓** | 将选中项加入播放列表（重复 URL 自动跳过） |
| **🗑 删除** | 从搜索结果中删除选中项 |
| **🎬 MV播放** | 在 MV 弹窗中播放选中项 |
| **分辨率** | MV 播放/保存的最大分辨率（HD / FHD / QHD / 2K / 4K） |
| **在浏览器中打开** | 用默认浏览器打开 YouTube |
| **双击** | 歌曲 → 加入列表 / MV → MV 播放 |

## 播放列表与播放

通过 **添加 ↓** 将搜索结果加入列表，可无限累积、多次搜索混合播放。

| 按钮 | 功能 |
|------|------|
| **▶ 播放** | 播放选中项（🎵 音频 / 🎬 MV 弹窗） |
| **⏸ 暂停** | 播放 ↔ 暂停 |
| **⏹ 停止** | 停止播放 |
| **⏭ 下一首** | 播放下一首（随机开启时随机选曲） |
| **🔀 随机播放** | 开启随机并立即随机播放 |
| **随机: 关/开** | 切换自动下一首是否随机 |
| **🗑 删除** | 从列表删除选中项 |
| **🗑 全部删除** | 清空整个列表 |
| **复制全部 URL** | 将列表 URL 复制到剪贴板 |

带 **▶** 的行表示当前正在播放。

## 音乐视频（MV）

MV 在**独立弹窗**中播放（初始大小 800×600）。

| 项目 | 说明 |
|------|------|
| 分辨率 | 在搜索结果旁选择 HD–4K（默认 FHD） |
| 全屏 | **F11** 或双击视频 |
| 关闭 | **Esc**（全屏时先退出全屏） |
| 说明 | 高分辨率通过 ffmpeg 合并音视频流 |

播放 MV 时，主窗口音频会自动停止。

## 下载 MP3 / MV

| 按钮 | 功能 |
|------|------|
| **⬇ MP3下载 (全部)** | 将播放列表全部歌曲保存为 MP3（192kbps） |
| **⬇ 保存选中歌曲** | 仅保存选中的一首 MP3 |
| **⬇ 保存选中 MV** | 保存选中的 MV 为 MP4（按所选分辨率） |
| **⬇ MV下载 (全部)** | 批量保存列表中全部 MV |

保存时会弹出文件夹选择对话框，底部状态栏显示进度。

---

## 本地 MP3

1. 点击 **📁 选择文件夹** 指定 MP3 目录
2. 选中歌曲后 **▶ MP3播放** 或双击播放
3. **全部添加** 可一次性加入播放列表

本地文件显示为 **💾 本地**。

---

## 歌词面板

右侧 **歌词** 面板显示当前播放歌曲的歌词（使用 `syncedlyrics`）。

- 开始播放后自动搜索
- 未找到时显示「未找到歌词。」
- 未安装 `syncedlyrics` 时歌词功能不可用

## 快捷键

| 按键 | 动作 | 范围 |
|------|------|------|
| `Enter` | 执行搜索 | 主窗口 |
| `F11` | 切换全屏 | MV 弹窗 |
| `Esc` | 退出全屏 / 关闭弹窗 | MV 弹窗 |

## 故障排除

| 问题 | 解决方法 |
|------|----------|
| 无法搜索 | 运行 `.venv/bin/pip install -U yt-dlp` |
| 无法播放 | 检查 VLC 是否安装 |
| MP3/MV 保存失败 | 安装 ffmpeg |
| 无歌词 | 运行 `pip install syncedlyrics` |
| 部分视频失败 | 可能受地区或 YouTube 限制 |
